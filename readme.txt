== Description ==

SMiLE Web is a free WordPress theme optimized for SEO, designed to help you gain visibility and rank higher in search engines quickly. This theme is fully configured, easy to self-manage, and integrates seamlessly with the WordPress Gutenberg block editor. Ideal for bloggers, small businesses, and professionals looking for a customizable, user-friendly theme.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
